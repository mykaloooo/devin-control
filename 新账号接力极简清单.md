## 1. 连 GitHub（必须 owner 操作）

> 一个 GitHub 组织同时只能挂一个 Devin 账号，所以先断旧、再连新。

1. **旧账号断开**：旧 Devin → Settings → Integrations → GitHub → `kaka-media` → **Disconnect**。
   - 只解除「Devin↔GitHub」连接，**不动仓库 / 团队 / 代码**。
2. **新账号连接**：新 Devin → Settings → GitHub → **Connect / Add connection** → 跳 GitHub → 选 **kaka-media** → **All repositories** → 由 **mykaloooo** 点 **Install / Approve**。

---

- 存在仓库 `kaka-media/2026IP` 的 `.devin/skills/` 里。

- 现有 skills：`repo-workflow`、`2026ip-main-control`。
